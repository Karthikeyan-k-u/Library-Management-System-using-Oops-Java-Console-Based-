# Library Management System (Java)

## 📌 Overview

This is a console-based Library Management System developed using Java. It allows users to manage books efficiently through a menu-driven interface.

---

## 🚀 Features

* Add new books with ID, title, and author
* Remove books from the library
* Search books using partial title match
* Issue and return books
* Display all books with status
* Input validation for better user experience

---

## 🛠️ Technologies Used

* Java
* Collections (ArrayList)
* Object-Oriented Programming (OOP)
* Scanner (for user input)

---

## 📂 Project Structure

* `Book` → Class representing book details
* `Library` → Class handling operations
* `Fulllength1.java` → Main class with menu system

---

## ▶️ How to Run

1. Open Command Prompt / Terminal
2. Compile the program:

   ```bash
   javac Fulllength1.java
   ```
3. Run the program:

   ```bash
   java Fulllength1
   ```

---

## 📊 How It Works

* The program displays a menu with options
* User selects an operation (add, remove, search, etc.)
* Inputs are taken using Scanner
* Actions are performed using Library class methods
* Results are displayed in the console
